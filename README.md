# Echoes 🎵

Turn your memories into cinematic songs.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` and add your OpenAI API key:
```
OPENAI_API_KEY=your_key_here
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## Features

- 🎵 AI-generated songs from personal memories
- 📝 Custom lyrics based on your story
- 🎨 Album cover visualization
- 📚 Personal memory library
- 💫 Shareable memory cards
